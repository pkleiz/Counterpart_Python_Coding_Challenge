Welcome

Welcome to the earthquake distance checker! This web application allows you to check the distance between a city and the nearest earthquake. The software uses HTML for the frontend and Django REST Framework for the backend. It's also has automatic detection of the latitude and longitude of the city, making it easier for you to use. Give it a try and stay safe!
